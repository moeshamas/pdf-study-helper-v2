# PDF Study Helper — Scope

## What the app does
PDF Study Helper ingests a small set of PDF documents, extracts and chunks their text into curated JSON files, and answers a user’s question using retrieved source chunks from those processed files.

## Supported tasks
- Ask a question about the processed PDFs
- Get a concise grounded answer
- View source chunks used for the answer

## Out of scope
- Uploading files through the browser
- User accounts or authentication
- Large-scale search across many document collections
- OCR-heavy scanned PDFs
- General web search
- Chat history and conversations
- Multi-user collaboration

## Why this scope
This is a narrow proof of concept designed to demonstrate ingestion, ETL, storage, reasoning, UI, testing, and deployment.
